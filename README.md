# GPA Calculator For All Pakistan Universities

A clean, responsive GPA & CGPA calculator built for students across Pakistan universities.

## Features

- Calculate semester GPA on a 4.0 scale
- Optional CGPA calculation using previous completed credit hours
- Support for multiple universities across Pakistan
- Subject-wise grade entry with credit hours
- Automatic grade point calculation
- Result cards for GPA and CGPA
- Charts for grade points and grade distribution
- Printable results page
- Mobile-friendly layout

## How to Run Locally

1. Download or clone this repository.
2. Open `index.html` in your browser.

No installation is required.

## Deploy on GitHub Pages

1. Push this project to a GitHub repository.
2. Open the repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Save.
6. Your site will be published at:

```text
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```
